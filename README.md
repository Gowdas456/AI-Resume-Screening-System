# AI-Resume-Screening-System

### Python Flask Application

**Tech Stack:** Python, Flask, PDF Processing, NLP, HTML, CSS

**Overview:** An AI-powered resume screening system designed to automate the recruitment process by extracting resume data, identifying candidate skills, and evaluating profiles based on job requirements. The system improves HR efficiency by reducing manual resume screening efforts.

**How it Works:** The application allows HR/Admin users to upload candidate resumes in PDF format. The system extracts text from resumes, identifies technical skills using skill extraction logic, and compares them with required job skills. Based on the matching percentage, it generates a candidate score, selection status, and ranking list for efficient hiring decisions.
